# Movie DB

movieDB is an application that provides services to find out the latest and trending movies. using https://www.themoviedb.org/ API

## Build with

[![Kotlin](https://img.shields.io/badge/kotlin-lang-blue)](https://kotlinlang.org/)

## Requirements

- Android Studio
